# Alraqabi Astro Starter

قالب عربي احترافي مبني باستخدام Astro، يدعم RTL وMarkdown وDecap CMS، ومهيأ للنشر على Cloudflare Pages.

## المميزات

- تصميم عربي RTL
- متجاوب مع الجوال
- Astro Content Collections
- مدونة وخدمات ومشاريع
- Decap CMS
- SEO وSchema
- Sitemap وRSS
- صفحات تصنيفات ووسوم
- صفحة 404
- أداء مرتفع
- دعم Cloudflare Pages

## التشغيل محليًا

```bash
npm install
npm run dev
